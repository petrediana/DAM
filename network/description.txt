View Data:
	-> first tap on the lateral menu to see all your options
	-> the first page shows you all the avalible stations which belong to Artery 1
	-> you can see all the subway arteries with each station
	-> you can manually add a station
	-> you can seek paths between two stations

	Arteries:
		-> see stations accordingly to their Artery
		-> I chose the most common used arteries in Bucharest	

	Add Node:
		-> here you can manually add a Station
		-> you can't add empty spaceses or null information, sorry :(

	Seek Paths:
		-> seek a path from a station to another
		-> you can see the possible routes and decide which one is more convenient for you
